Requirements:
For this programme to run the local machine must have installed python 3.0 or above.

How to run the code:
Import the code to a compiler and then write the command  python3 followed by name and location of file in the 
terminal and then press enter. It will run the code.

This code ask the user to choose from 4 options i.e., clustering by k_means without Normalisation, clustering by
k_means with Normalisation, clustering by k_medians without Normalisation and clustering by k_medians with Normalisation.
Depending upon the choice it clusters the loaded data into 9 different set of clustering according to choosen
criteria, evaluate the clustering by B Cubed criteria and plot precision, recall, f-score vs number of clustering.
